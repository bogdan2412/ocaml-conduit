Library conduit-mirage

This library exposes the following toplevel modules:

  • Conduit_mirage Functorial connection establishment interface that is compatible with the Mirage libraries.
  • Conduit_xenstore Establish Vchans via named endpoints in XenStore
  • Resolver_mirage Functorial interface for resolving URIs to endpoints.
