Library conduit-lwt-unix

This library exposes the following toplevel modules:

  • Conduit_lwt_launchd
  • Conduit_lwt_server
  • Conduit_lwt_tls TLS/SSL connections via OCaml-TLS
  • Conduit_lwt_unix Connection establishment using the Lwt_unix library
  • Conduit_lwt_unix_ssl TLS/SSL connections via OpenSSL C bindings
  • Resolver_lwt_unix Resolve URIs to endpoints using Unix system calls
